# API Health Monitor

A real-time dashboard to monitor the health, latency, and uptime of public APIs.

## Features

- Monitor multiple API endpoints
- Real-time health checks with latency tracking
- Visual status indicators (UP, SLOW, DOWN)
- Health score system
- Add/remove services dynamically

## Getting Started

### Prerequisites
- Node.js 18+

### Installation

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the dashboard.

### Build & Deploy

```bash
npm run build
npm start
```

## Tech Stack

- **Framework:** Next.js 14+
- **Styling:** Tailwind CSS
- **Storage:** JSON (File-based)
